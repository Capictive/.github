# .github

This is the special `.github` repository for the Capictive organization.

## Organization Profile

The `profile/README.md` file in this repository defines the public profile that appears on our GitHub organization page.

## Purpose

This repository serves as a central location for:
- Organization profile README
- Community health files (templates, guidelines, etc.)
- Shared workflows and configurations
